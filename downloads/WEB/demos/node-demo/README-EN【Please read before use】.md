# Based on the imou-player Node Version

## Preparation before operation

### 1. Install dependencies

```javascript
npm install
```

### 2. Copy all files from the root directory to the dist folder

> WasmLib/ 、imou-player.js、 imou-player.css、indexEn.js、indexEn.html

### 3. Run

```javascript
node index.js
```
